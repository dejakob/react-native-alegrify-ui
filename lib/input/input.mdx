## Properties


|Property |Description            |Required? |Type    |
|---------|-----------------------|----------|--------|
|defaultValue |Default value of the input |No |String |
|value |Value of the input |No |String |
|label |Label to show above the input |No |String |
|type |phone,number,email,text,password,range |No |String |
|multiline |Multiple lines on input? |No |Boolean |
|onBlur |Event on blur |No |Function |
|onFocus |Event on focus |No |Function |
|onChangeText |Event on text change |No |Function |
|onValueChange |Event on value change (range input) |No |Function |
|error |Error message to show bellow input |No |String |
